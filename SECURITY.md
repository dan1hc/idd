| Version | Supported          |

## Reporting a Vulnerability

Please use this repository's `security/advisories` interface directly to report a vulnerability.

Project maintenance will analyze the situation and respond within 24-48 hours, depending on severity.

If a vulnerability is confirmed, it will be patched in a time period commensurate with severity.
